hexo 基本使用

创建新博客

  • 在\source\_posts下创建一个新的Markdown文件，可以通过Typora编辑该md文件
  • 通过Typora直接创建新文件，保存到\_posts文件夹下即可

启动博客

  • 通过该命令返回的端口地址，即可在浏览器对生成网页进行预览
  • 该命令可简写hexo s

生成静态文件夹

  • 生成存储于\public静态html 文件
  • 简写hexo g

远程部署

  • 通过git 将网页上传到github上，实现博客功能
    • 前提：修改

hexo使用记录

Date 10/10/2017 hexo server 启动后主页打不开

描述：启动hexo-s 后正常显示Hexo is running at http://localhost:4000/但是浏览器打不开

原因：福昕阅读软件占用端口

解决：

  1. 查看端口占用情况 ，命令行输入netstat -ano|findstr 4000
  2. 查看占用端口的程序PID
  3. 打开任务管理器，查看详细，关闭对应PID
